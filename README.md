# E-Commerce Relational Database Order Analytics & SQL Query Pipeline

## Project Overview
This repository holds a modular production-grade SQL script portfolio deployed to query, transform, and isolate growth metrics from transactional relational database frameworks. Utilizing advanced analytical commands, nested conditional workflows, and time-series aggregation functions, this pipeline processes raw structural backend logs into refined corporate metrics ready for direct business visualization.

## Data Engineering & Query Architecture
The structured script layouts are engineered to execute advanced analytical computations across key operational layers:

* **Macro KPI Summary:** Uses SUM, COUNT, and AVG aggregate functions to isolate total enterprise gross processing value ($1,264,761.96), distinct volume count (1,200), and consolidated transaction averages ($1,052.55).
* **Product Contribution:** Employs GROUP BY and ORDER BY logic to rank exact categorical inventory distribution weightings, proving Chairs and Printers lead individual totals.
* **Order Status Analytics:** Incorporates percent-of-total nested subqueries to expose significant corporate risk boundaries by quantifying cancelled orders at 20.8% and returned items at 20.6%.
* **Financial Gateway Share:** Applies multi-variable partition summaries to monitor payment preferences, revealing Credit Cards ($263,847.63) and Online Banking ($262,442.94) as the dominant transactional channels.
* **Acquisition Efficiency:** Tracks source categorization filters to map operational marketing parameters, verifying Instagram as the highest-performing digital channel ($275,285.45).
* **Promotion Performance:** Implements CASE WHEN conditional transformations to standardize database NULL entities into a uniform string architecture, isolating the FREESHIP incentive as the highest volume promo driver.
* **Chronological Pacing:** Leverages date formatting string functions to group localized timestamps into structured monthly buckets to map multi-year enterprise cyclical trends.
